# 👾 Imposter Who

**Imposter Who** is a fun multiplayer deduction game where players must figure out **who the imposter is before it's too late**. Work together, watch for suspicious behaviour, and vote wisely… because the imposter is hiding among you.

---

## 🎮 Gameplay

1. Players join a game lobby.
2. Each round, **one player becomes the Imposter**.
3. Normal players receive a **secret word**.
4. The Imposter does **not know the word** and must bluff.
5. Players take turns giving **clues related to the word**.
6. After the clues, everyone **votes for who they think the Imposter is**.
7. If the Imposter is caught, the crew wins.  
   If not, the Imposter survives to the next round.

---

## 🧠 Strategy Tips

- Give clues that are **clear to others but not obvious to the Imposter**.
- Watch for players giving **vague or strange clues**.
- The Imposter should try to **blend in and guess the word**.

## 🚀 Getting Started

1. Download or clone the repository.
2. Install dependencies (if required).
3. Run the game server or open the game.
4. Invite friends and start playing!

Example:

```bash
git clone https://github.com/yourusername/imposter-who.git
cd imposter-who
npm install
npm start
```

---

## 📜 Rules Summary

- One **Imposter** per round.
- All other players know the **secret word**.
- Everyone gives **one clue each round**.
- Players **vote after clues are given**.
- Majority vote decides who is eliminated.
